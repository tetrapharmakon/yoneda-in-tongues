\tran{Bahasa Melayu (ISO 639-3 msa)}{Lema Yoneda}{
	Biar $\mathcal C$ menjadi sebuah kategori kecil dan $F$ sebuah fungtor daripada kategori ini ke kategori set. Maka, untuk setiap objek $X$ daripada $\mathcal C$ wujud bijeksi tabii antara set transformasi tabii $\hom(-,X)\Rightarrow F$ dan set $FX$, dan isomorfisma ini ditakrifkan oleh korespondensi
	\[
		\Big(\xi : \hom(-,X)\Rightarrow F\Big) \mapsto \xi_X(1_X)
	\]
	(fungsi ini adalah bijektif).
}
