\tran{Íslenska (ISO 639-3 isl)}{Hjálparsetning Yoneda}{
	Látum $\mathcal C$ vera lítinn flokk og $F$ fallvörpun frá þessum flokki í flokk mengjanna. Þá er til, fyrir sérhvern hlut $X$ í $\mathcal C$, náttúruleg gagntæk vörpun milli mengjar náttúrulegra ummyndana $\hom(-,X)\Rightarrow F$ og mengjarinnar $FX$, og þessi einsmótun er skilgreind með samsvöruninni
	\[
		\Big(\xi : \hom(-,X)\Rightarrow F\Big) \mapsto \xi_X(1_X)
	\]
	(þessi vörpun er gagntæk).
}
