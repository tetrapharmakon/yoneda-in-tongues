\tran{Norrǿnt mál (ISO 639-3 non)}{Hjálparsetning Yoneda}{
	Sé $\mathcal C$ lítill flokkr ok $F$ vörpun frá þessum flokki í flokk mengjanna. Þá er til, fyri hvern hlut $X$ í $\mathcal C$, náttúruleg gagntæk vörpun milli mengjar náttúrulegra ummyndana $\hom(-,X)\Rightarrow F$ ok mengjarinnar $FX$, ok þessi einsmótun er greind með samsvöruninni
	\[
		\Big(\xi : \hom(-,X)\Rightarrow F\Big) \mapsto \xi_X(1_X)
	\]
	(þessi vörpun er gagntæk).
}
