\tran{Scots (ISO 639-3 sco)}{Yoneda lemma}{
	Lat $\mathcal C$ be a wee category, an $F$ a functor frae this category tae the category o sets. Than, for ilka object $X$ o $\mathcal C$ thare is a naitural bijection atween the set o naitural transformations $\hom(-,X)\Rightarrow F$ an the set $FX$, an this isomorphism is defined bi the correspondence
	\[
		\Big(\xi : \hom(-,X)\Rightarrow F\Big) \mapsto \xi_X(1_X)
	\]
	(this function is bijective).
}
