\tran{Cymraeg (ISO 639-3 cym)}{Lema Yoneda}{
	Boed $\mathcal C$ yn gategori fach ac $F$ yn ffwnctor o'r categori hwn i gategori'r setiau. Yna, ar gyfer pob gwrthrych $X$ o $\mathcal C$ mae bijecsiwn naturiol rhwng y set o drawsffurfiadau naturiol $\hom(-,X)\Rightarrow F$ a'r set $FX$, ac mae'r isomorffedd hwn wedi'i ddiffinio gan y cydweddiad
	\[
		\Big(\xi : \hom(-,X)\Rightarrow F\Big) \mapsto \xi_X(1_X)
	\]
	(mae'r ffwythiant hwn yn fijectiw).
}
