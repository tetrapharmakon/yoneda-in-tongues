\tran{Español (ISO 639-3 spa)}{Lema de Yoneda}{
	Sea $\mathcal C$ una categoría pequeña y $F$ un funtor de esta categoría en la categoría de conjuntos. Entonces, para todo objeto $X$ de $\mathcal C$ existe una biyección natural entre el conjunto de transformaciones naturales $\hom(-,X)\Rightarrow F$ y el conjunto $FX$, y este isomorfismo está definido por la correspondencia
	\[
		\Big(\xi : \hom(-,X)\Rightarrow F\Big) \mapsto \xi_X(1_X)
	\]
	(esta función es biyectiva).
}
