\tran{Čeština (ISO 639-3 ces)}{Yonedovo lemma}{
	Nechť $\mathcal C$ je malá kategorie a $F$ funktor z této kategorie do kategorie množin. Pak pro každý objekt $X$ kategorie $\mathcal C$ existuje přirozená bijekce mezi množinou přirozených transformací $\hom(-,X)\Rightarrow F$ a množinou $FX$, a tento izomorfismus je definován korespondencí
	\[
		\Big(\xi : \hom(-,X)\Rightarrow F\Big) \mapsto \xi_X(1_X)
	\]
	(tato funkce je bijektivní).
}
